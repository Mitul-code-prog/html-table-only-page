# html-table-only-page
A pure HTML blog layout built with table tags, with using lorem Picsum as structural placeholders.
A classic web development where I built a complete webpage layout using only " HTML table tag"

## Live demo
[view live](https://mitul-code-prog.github.io/html-table-only-page/) 

## Tech used
Pure HTMl5( utilizing <table>,<td>,<tr>,<img> tag)
To make webpage look better used backgroundcolor to remove the border of table

## How to view
1.Clone or download this repository.
2.open the index.html file directly in any morden web browser. 
3.or [view live](https://mitul-code-prog.github.io/html-table-only-page/)
